# Чек-лист ручного тестирования КБЖУ

Тестировщик: erdmxp  
Разработчик модуля: MudryMistychesky  
Дата проверки: 11 мая 2026 года

- [ ] Страница КБЖУ открывается из главного меню.
- [ ] Фон, заголовок, форма и итоговые показатели отображаются.
- [ ] По умолчанию выбрана текущая дата.
- [ ] Можно выбрать другую дату.
- [ ] Поиск продукта начинается после ввода названия.
- [ ] Результаты поиска содержат название и показатели КБЖУ.
- [ ] Продукт можно выбрать из результатов поиска.
- [ ] После выбора отображаются калории, белки, жиры и углеводы.
- [ ] Можно указать массу продукта в граммах.
- [ ] Запись добавляется в дневник выбранного дня.
- [ ] Значения пересчитываются пропорционально массе.
- [ ] Итоги дня пересчитываются после добавления.
- [ ] Запись можно удалить.
- [ ] Итоги дня пересчитываются после удаления.
- [ ] Нулевая и отрицательная масса не принимаются.
- [ ] Пустой продукт не добавляется.
- [ ] Записи разных дат не смешиваются.
- [ ] Данные разных пользователей разделены.
- [ ] После обновления страницы записи сохраняются.
- [ ] После перезапуска сервера записи загружаются из PostgreSQL.
