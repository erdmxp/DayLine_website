# Чек-лист ручного тестирования финансов

Тестировщик: erdmxp  
Разработчик модуля: MudryMistychesky  
Дата проверки: 11 мая 2026 года

- [ ] Страница финансов открывается из главного меню.
- [ ] Отображаются форма, список операций и итоговые карточки.
- [ ] Можно выбрать тип «Доход».
- [ ] Можно выбрать тип «Расход».
- [ ] Список категорий меняется в зависимости от типа.
- [ ] Можно указать сумму, дату и описание.
- [ ] Доход сохраняется и появляется в списке.
- [ ] Расход сохраняется и появляется в списке.
- [ ] Общий доход рассчитывается правильно.
- [ ] Общий расход рассчитывается правильно.
- [ ] Баланс равен доходам минус расходы.
- [ ] Фильтр по месяцу работает.
- [ ] Фильтр по типу операции работает.
- [ ] Фильтр по категории работает.
- [ ] Операцию можно удалить.
- [ ] После удаления пересчитываются итоги.
- [ ] Нулевая и отрицательная сумма не принимаются.
- [ ] Обязательные поля нельзя оставить пустыми.
- [ ] Операции разных пользователей разделены.
- [ ] Данные сохраняются после обновления и перезапуска сервера.
