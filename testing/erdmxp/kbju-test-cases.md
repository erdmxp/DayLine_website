## KBJU-01 — открытие дневника

- Приоритет: высокий.
- Предусловие: пользователь авторизован.
- Шаги: открыть главную страницу; нажать «Питание»; проверить форму, список и блок итогов.
- Ожидаемый результат: страница открылась; выбрана текущая дата; итоговые значения равны данным выбранного дня.
- Фактический результат: ____________________
- Статус: PASS / FAIL

## KBJU-02 — поиск продукта

- Приоритет: критический.
- Тестовые данные: `Молоко`.
- Шаги:
  1. Ввести название в строку поиска.
  2. Подождать появления результатов.
  3. Проверить название и показатели продукта.
- Ожидаемый результат: показаны подходящие продукты с калориями, белками, жирами и углеводами.
- Фактический результат: ____________________
- Статус: PASS / FAIL

## KBJU-03 — добавление продукта

- Приоритет: критический.
- Предусловие: найден и выбран продукт «Молоко».
- Тестовые данные: масса `200` граммов, дата 11.05.2026.
- Шаги:
  1. Выбрать продукт.
  2. Указать массу 200 г.
  3. Нажать кнопку добавления.
  4. Обновить страницу.
- Ожидаемый результат: запись появилась в выбранном дне; КБЖУ рассчитано для 200 г; запись сохранилась после обновления.
- Фактический результат: ____________________
- Статус: PASS / FAIL

## KBJU-04 — пересчёт итогов

- Ожидаемый результат: итоговые калории, белки, жиры и углеводы равны сумме показателей записей.
- Фактический результат: ____________________
- Статус: PASS / FAIL
