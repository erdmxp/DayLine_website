# Чек-лист ручного тестирования списка дел

Тестировщик: MudryMistychesky  
Разработчик модуля: erdmxp  
Дата проверки: 11 мая 2026 года

Перед проверкой запустить `npm run dev` и `npm run StartServ`, войти в тестовый аккаунт и открыть страницу списка дел.

## Открытие и навигация

- [ ] Страница списка дел открывается из главного меню.
- [ ] Фоновое изображение и элементы интерфейса отображаются.
- [ ] Календарь показывает текущий месяц и год.
- [ ] Кнопки предыдущего и следующего месяца работают.
- [ ] Можно выбрать конкретный день календаря.
- [ ] Выбранная дата визуально отличается от остальных.

## Добавление задач

- [ ] Кнопка добавления открывает поле ввода.
- [ ] Новая задача сохраняется на выбранную дату.
- [ ] После добавления задача появляется без обновления страницы.
- [ ] Счётчик задач изменяется.
- [ ] Длинный текст не ломает карточку.
- [ ] Пустая задача не должна сохраняться.

## Изменение и удаление

- [ ] Задачу можно выделить кликом.
- [ ] Выделенная задача визуально отмечается.
- [ ] Текст выбранной задачи можно изменить.
- [ ] Изменение сохраняется после обновления страницы.
- [ ] Задачу можно удалить отдельной кнопкой.
- [ ] После удаления обновляются список и счётчик.

## Данные и доступ

- [ ] При смене даты показываются задачи только выбранного дня.
- [ ] После возвращения на дату сохранённые задачи остаются.
- [ ] Задачи одного пользователя не видны другому.
- [ ] API задач без авторизации возвращает отказ в доступе.
- [ ] После перезапуска сервера данные остаются в PostgreSQL.
