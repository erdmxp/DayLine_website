# DayLine Website

Новая поэтапная разработка учебного веб-приложения DayLine.

## Планируемый локальный запуск

- Node.js — сервер приложения;
- PostgreSQL — локальная база данных;
- браузер — клиентская часть.

Проект создаётся по этапам с проверкой каждого этапа перед продолжением.

## Умный планировщик

На странице списка дел работает мини-чат: помощник уточняет цель, предлагает идеи и позволяет добавлять подходящие задачи на выбранную дату.

Для работы чата добавьте в Render переменные:

```env
OPENROUTER_API_KEY=ваш_ключ
AI_MODEL=openrouter/free
AI_BASE_URL=https://openrouter.ai/api/v1
APP_URL=https://day-line.ru
```

Ключ хранится только на сервере и не передаётся в браузер. Конкретную бесплатную модель Qwen можно указать в `AI_MODEL` с суффиксом `:free`.

Если ключ отсутствует или провайдер временно недоступен, интерфейс показывает понятную ошибку и не подставляет шаблонные задачи.
