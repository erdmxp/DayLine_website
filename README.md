# DayLine

DayLine — сайт для планирования задач, контроля питания и учёта личных финансов.

Над проектом работали:

- **Эрдэм (erdmxp)** — главная страница, календарь и список дел;
- **Валентин (MudryMistychesky)** — регистрация, авторизация, питание, КБЖУ и финансы.

Рабочая версия: [day-line.ru](https://day-line.ru)
